之前曾经发过帖子，其中也有关于思考时间的建议。这次想提个更详细的方案： [b]将思考时间改成“动态”。[/b] 即时间不再是固定的1分半，而是依据选项的多少计算。选项是指手中卡牌，场上随从，武器和技能的合计数量。 举例来说： 1. 牌局末期，手中无卡牌，场上无随从，无武器，仅有英雄技能可用。 那么思考时间 = 10秒 x 1 = 10秒 2. 牌局中期，手中3卡牌，场上2随从，有武器，有技能。 那么思考时间 = 10秒 x (3 + 2 + 1 + 1) = 70秒 3. 刚刚开局，手中4卡牌，无随从，无武器，有技能。 那么思考时间 = 10秒 x (4 + 1) = 50秒 注1：10秒这个时间单位只是个例子。 注2：简化掉许多细节。 比如，开局时手中卡牌都是大于1水晶，根本无法出手，但仍然给足时间。毕竟仍然需要考察考量。 再比如，手中有着大量卡牌，场上也有许多随从，排列组合起来选择很多，但仍然还是10秒时间。占优势的情况下，这个设置还是可以接受的吧。 希望这个建议能够帮助改善炉石传说的游戏环境。
